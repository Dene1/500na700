## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Запуск JSON Server (данные новостей) - должен работать на порту 3002
npm run server

# Сборка проекта
npm run build

## Структура проекта
- `src/` - исходный код React приложения
- `db.json5` - данные для JSON Server
- `package.json` - зависимости и скрипты

## Зависимости
- React + Vite
- JSON Server для моковых данных
- react-number-format для маски номера
